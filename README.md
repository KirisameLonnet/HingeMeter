# HingeMeter

只是一个最最简单的安卓折叠手机铰链角度指示器，仿照 Apple 水平仪 UI。

# 使用说明

- 进入应用后会显示当前铰链角度（需设备支持 `TYPE_HINGE_ANGLE` 传感器），数值直接取传感器数据，不做平滑。
- 双击画面可选择并插入 GIF 贴图，贴图动画会随铰链角度变化推进。
- 贴图可拖动位置；双指可缩放与旋转。
- 多个贴图时，最后编辑的贴图位于最上层。
- 长按画面中心可开启/关闭转轴声音模拟（默认开启），声音强度/音高随开合速度变化，屏幕会短提示 `Sound on/off`。
- 长按贴纸可在贴纸中心出现删除 X，点击删除；删除时有弹性动画与震动反馈。
